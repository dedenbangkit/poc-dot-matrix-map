# Dot Matrix Map - West Java Agricultural Data

An interactive choropleth and dot matrix map visualization of agricultural data for West Java, Indonesia. Built with React, Vite, and Leaflet.

## Features

- **Choropleth visualization** for basic statistics and crop data
- **Dot matrix visualization** for farming methods distribution
- Interactive hover tooltips with detailed regional information
- Multiple indicator types: Basic Statistics, Crops Grown, Farming Methods

## Tech Stack

- React 19
- Vite
- Leaflet / React-Leaflet
- Turf.js (geospatial analysis)
- TopoJSON

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
